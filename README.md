# repo-main
Main repository for Unyw wrappers.  Install them from the [unyw.apk](https://unyw.github.io/android-apk/unyw.apk).

`

`

## Build
LINUX  REQUIRED
Make sure that `wget` and `proot` are installed in your system.
Run 
```bash
node build.js packages/PACKAGE-NAME
```
